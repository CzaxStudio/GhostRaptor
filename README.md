# GhostRaptor

Ghost Raptor is an easy to use hacking tool for people new in hacking.

## Get started

Make sure you have **git** and **bash** installed on your system.

```bash
git clone https://github.com/CzaxStudio/GhostRaptor
# After this, enter the folder GhostRaptor is cloned to
bash ghost.sh
```

Tools Included

Once launched, you can easily install:

    Phishing (zphisher)

    Webcam Hacking (CamPhish)

    DDoS Testing (DDoS-Ripper)

    Virus Creation (TigerVirus)

    BadMod

After this you select tool name and download the tools for free.
